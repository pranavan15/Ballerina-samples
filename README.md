# Ballerina-samples
  Samples written using Ballerina language. These use-case samples explain how we can use the Ballerina language to 
  write programs to perform many tasks easier than many other languages. 

# How to deploy samples
  1. Go to http://www.ballerinalang.org and click Download.

  2. Download the Ballerina Tools distribution and unzip it on your computer. Ballerina Tools includes the Ballerina 
  runtime plus the visual editor (Composer) and other tools.
  
  3. Add the <ballerina_home>/bin directory to your $PATH environment variable so that you can run the Ballerina 
  commands from anywhere.

  4. After setting up <ballerina_home>, run: `$ ballerina run SAMPLE-NAME.bal` with appropriate configurations 
  specified.
